## Asset Management Notes

Following the [Symphony Core Documentation Architecture](./../symphony-core-documentation-architecture.md) best practice:
- Large files (images, PDFs) are stored in Google Drive
- This registry maintains links and metadata
- Each asset includes: location, purpose, status, last updated date

**To add new print media assets:**
1. Upload file to Google Drive
2. Get shareable link
3. Add entry to this registry
4. Reference from relevant document
