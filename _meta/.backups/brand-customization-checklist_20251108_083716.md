# Symphony Core Brand Customization Checklist

**Complete checklist for applying Symphony Core branding to Extendly presentation**

Use this checklist to ensure consistent branding across all presentation materials.

---

## Overview

**Purpose:** Apply Symphony Core brand identity to Extendly Small Business Snapshot presentation template

**Source Template:** `presentations/master/symphony-core-automation-demo-master.pptx`

**Brand Guide:** `02-marketing-brand/brand/brand-guidelines/symphony_core_web_style_guide.md`

**Estimated Time:** 2-3 hours (first-time customization)

---

## Phase 1: Visual Branding

### Colors

- [ ] **Primary Accent Color Updated**
  - Current Extendly color identified
  - Replaced with Symphony Blue: `#3682AEFF`
  - Applied to all slides
  - Verified consistency across presentation

- [ ] **Text Color Updated**
  - Headings changed to Core Slate: `#1C242BFF`
  - Body text changed to Core Slate: `#1C242BFF`
  - Light text on dark backgrounds verified (white or Symphony Light)

- [ ] **Background Colors Updated**
  - Section backgrounds changed to Symphony Light: `#B8D4F1FF`
  - Alternating backgrounds for visual variety
  - White backgrounds where appropriate

- [ ] **CTA/Button Colors Updated**
  - Call-to-action buttons: Success Accent `#2563EBFF`
  - Hover states: Symphony Blue `#3682AEFF`
  - Sufficient color contrast verified (accessibility)

- [ ] **Color Consistency Check**
  - All slides reviewed for color consistency
  - No Extendly brand colors remaining
  - Color palette limited to SC approved colors
  - Screenshots/images complement color scheme

### Typography

- [ ] **Heading Fonts Updated**
  - H1 (page titles): Roboto Bold, 48px
  - H2 (section headings): Roboto Bold, 36px
  - H3 (subsections): Roboto Medium, 28px
  - All headings consistent across slides

- [ ] **Body Text Fonts Updated**
  - Primary body: Inter Regular, 18px
  - Secondary body: Inter Regular, 16px
  - Bullet points: Inter Regular, 16-18px

- [ ] **Small Text Fonts Updated**
  - Disclaimers: Source Sans Pro Regular, 14px
  - Footnotes: Source Sans Pro Regular, 12px
  - Footer text: Source Sans Pro Regular, 12px

- [ ] **Font Readability Check**
  - All text readable on projector
  - Sufficient size for room presentations
  - Line spacing appropriate
  - Not too much text per slide (max 6 bullet points)

### Logo & Branding

- [ ] **Title Slide Logo Added**
  - Symphony Core logo inserted
  - High-resolution PNG or SVG used
  - Properly sized (200-300px wide)
  - Positioned appropriately (typically center or top-right)

- [ ] **Master Slide Logo Added**
  - Logo added to slide master template
  - Appears on all slides (except title slide if preferred)
  - Smaller size (100-150px wide)
  - Positioned consistently (typically top-right or bottom-right corner)

- [ ] **Logo Quality Check**
  - No pixelation or blurriness
  - Proper aspect ratio maintained
  - Logo colors accurate (not distorted)
  - Background transparent (if applicable)

### Layout & Spacing

- [ ] **Consistent Margins**
  - All slides have consistent margins
  - Text not too close to edges
  - Adequate white space around elements

- [ ] **Visual Hierarchy**
  - Headings clearly distinguishable from body text
  - Important information emphasized (size, color, or weight)
  - Logical reading flow (left to right, top to bottom)

- [ ] **Alignment**
  - Text properly aligned (left, center, or right as appropriate)
  - Images and graphics aligned with text
  - Elements don't appear randomly placed

## Phase 3: Content Quality & Accuracy

### Proofread & Grammar

- [ ] **Spell Check Completed**
  - PowerPoint spell check run
  - All flagged items reviewed
  - Industry-specific terms added to dictionary

- [ ] **Grammar Reviewed**
  - Sentences complete and grammatically correct
  - Consistent tense throughout (present tense preferred)
  - Active voice used (not passive)

- [ ] **Terminology Consistent**
  - "Symphony Core" spelled consistently
  - "automation platform" vs "CRM" terminology consistent
  - Industry terms used correctly
  - Acronyms defined on first use

- [ ] **No Placeholder Text**
  - All [ENTER...] placeholders removed
  - All TODO notes removed
  - All sample/example text updated with real content

### Visual Quality

- [ ] **Images High-Resolution**
  - No blurry or pixelated images
  - All images minimum 150 DPI
  - Screenshots clear and readable
  - Stock photos (if used) high quality

- [ ] **Charts/Graphs Accurate**
  - Data accurate and sourced
  - Labels clear and readable
  - Legend included if needed
  - Colors follow brand palette

- [ ] **Icons & Graphics Consistent**
  - Icon style consistent throughout
  - Graphics complement (not distract from) content
  - File sizes optimized (presentation not too large)

- [ ] **Screenshots Updated**
  - No Extendly branding in screenshots
  - SC platform screenshots (if using actual screenshots)
  - Client data sanitized (if showing real examples)
  - Screenshots recent (not outdated UI)

### Pricing & Packages

- [ ] **Pricing Information Added (if applicable)**
  - Package tiers defined
  - Pricing accurate and current
  - Features clearly mapped to packages
  - Implementation fees mentioned

- [ ] **ROI/Value Calculations Accurate**
  - Metrics realistic (not exaggerated)
  - Sources cited if using statistics
  - Calculations correct
  - Comparisons fair and accurate

### Case Studies & Testimonials

- [ ] **Real Examples Added (if available)**
  - Client permission obtained
  - Client data sanitized/anonymized as needed
  - Results quantified (specific numbers)
  - Testimonial quotes accurate

- [ ] **Generic Extendly Examples Removed**
  - No Extendly client examples
  - No generic "Client X" placeholders
  - Industry-specific examples relevant to SC target market

## Phase 4: Technical Quality

### Animations & Transitions

- [ ] **Animations Appropriate**
  - Not too distracting or flashy
  - Support (not detract from) content
  - Consistent style throughout
  - Work properly in all slides

- [ ] **Transitions Consistent**
  - Same transition used throughout (or minimal variation)
  - Transition speed appropriate
  - No jarring or distracting transitions

### Links & Interactive Elements

- [ ] **Hyperlinks Work**
  - All links tested and functional
  - Links point to SC website (not Extendly)
  - Email links formatted correctly (mailto:)
  - Phone links formatted correctly (tel:)

- [ ] **Embedded Media Tested**
  - Videos play properly (if embedded)
  - Audio works (if included)
  - No broken media links

### File Properties

- [ ] **File Named Correctly**
  - Follows naming convention: `symphony-core-[type]-deck.pptx`
  - No spaces in filename (use hyphens)
  - Version number if applicable (v1.0, v2.0)

- [ ] **File Size Optimized**
  - Images compressed (if file size >20 MB)
  - Unused slides/layouts deleted
  - Embedded fonts if needed
  - File easily shareable (not too large for email)

- [ ] **Metadata Updated**
  - File properties: Author = "Symphony Core"
  - Title property set
  - Subject/description added
  - Keywords added for searchability

## Phase 5: Accessibility

### Color Contrast

- [ ] **Text Readable on Backgrounds**
  - Minimum 4.5:1 contrast ratio (AA standard)
  - Dark text on light backgrounds
  - Light text on dark backgrounds
  - No low-contrast combinations

- [ ] **Color Not Sole Indicator**
  - Important information not conveyed by color alone
  - Icons or labels supplement color coding
  - Accessible to colorblind viewers

### Alt Text

- [ ] **Images Have Alt Text**
  - All images have descriptive alt text
  - Alt text concise but informative
  - Decorative images marked as such

- [ ] **Charts Have Descriptions**
  - Chart data described in text or alt text
  - Key insights stated (not just visual)

### Readability

- [ ] **Font Sizes Appropriate**
  - Minimum 18px for body text
  - Minimum 24px for headings
  - No font smaller than 12px (footer/disclaimer)

- [ ] **Logical Reading Order**
  - Content flows logically
  - Screen reader would read in correct order
  - No confusing layout
