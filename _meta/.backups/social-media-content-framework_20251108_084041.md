# Social Media Content Standardization Framework

## Recommended Document Structure

You need a **three-tier system** for maximum effectiveness:

### 1. Master SOP (Standard Operating Procedure)
**Purpose:** Process documentation for training and consistency
**Format:** Comprehensive step-by-step guide

### 2. Platform Templates
**Purpose:** Quick-start frameworks for content creation
**Format:** Fill-in-the-blank structures with examples

### 3. Quick Reference Guide
**Purpose:** At-a-glance specifications and limits

### 1. Platform Specifications

#### LinkedIn Specifications
```markdown
## LinkedIn Post Requirements

### Character Limits
- Standard posts: 3,000 characters (including spaces, punctuation)
- Preview text: 200-220 characters (before "see more")
- Comments: 1,500 characters
- Headlines: ~220 characters
- Articles: 125,000 characters
- Emojis: Count as 2 characters each

### Optimal Structure
1. Hook (First 200 characters)
   - Must contain complete thought
   - Include key value proposition
   - End with intrigue or question

2. Body (Remaining 2,800 characters)
   - Use line breaks every 2-3 sentences
   - Include 3-5 main points
   - Add white space for readability

3. Call to Action
   - Clear next step
   - Question for engagement

4. Hashtags
   - 3-5 relevant hashtags
   - Mix broad and niche tags
```

### 2. Content Creation Process

```markdown
## Step-by-Step Process

### Phase 1: Planning
1. Identify content type (Educational/Case Study/Thought Leadership)
2. Define primary message
3. Select target platform(s)
4. Check platform-specific requirements

### Phase 2: Content Creation
1. Open appropriate template
2. Input to AI tool with context:
   - Company voice guidelines
   - Target audience
   - Platform specifications
   - Character limits
3. Generate initial draft
4. Review against checklist

### Phase 3: Optimization
1. Verify character count
2. Optimize hook (first 200 characters)
3. Add platform-specific elements
4. Include visual requirements

### Phase 4: Quality Check
- [ ] Within character limit?
- [ ] Hook compelling?
- [ ] Brand voice consistent?
- [ ] CTA clear?
- [ ] Hashtags relevant?
- [ ] Visual elements ready?
```

### Instagram Post Template
```markdown
# Instagram Post Template

[Hook - First sentence must capture attention]

[Story or value - 3-4 sentences]

[Bullet points or emoji-separated list]
• Point 1
• Point 2
• Point 3

[CTA with clear instruction]

## Hashtags (up to 30, use 10-15 optimal)
#hashtag1 #hashtag2 #hashtag3

- Format: Square (1080x1080) or Vertical (1080x1350)
- Text overlay: Max 20% of image
```

## Implementation Guide

### Setting Up the System

1. **Create Master Folder Structure:**
```
/Social Media Standards/
  ├── SOPs/
  │   └── Social_Media_Creation_SOP_v1.0.md
  ├── Templates/
  │   ├── LinkedIn_Template.md
  │   ├── Instagram_Template.md
  │   └── Facebook_Template.md
  ├── Quick References/
  │   └── Platform_Specs_QuickRef.pdf
  └── Examples/
      └── Successful_Posts/
```

2. **AI Tool Configuration:**
Create saved prompts/instructions in your AI tool that include:
- Platform specifications
- Character limits
- Brand voice guidelines
- Template structures

3. **Version Control:**
- Track changes in templates
- Update character limits quarterly
- Document what works/doesn't work

### Sample AI Instruction Set
```markdown
You are creating social media content for Symphony Core.

Platform Requirements:
- LinkedIn: 3,000 chars max, hook in first 200 chars
- Instagram: 2,200 chars max, first line critical
- Facebook: Keep under 500 chars for optimal engagement

Brand Voice:
- Professional yet approachable
- No jargon
- Results-focused
- Local business friendly

Always:
- Front-load value in first sentence
- Include clear CTA
- Suggest 3-5 relevant hashtags
- Indicate character count
- Provide both long and short versions
```

This standardization system ensures consistency while remaining flexible enough for different content types and platforms. The SOP provides comprehensive guidance, templates enable quick creation, and the quick reference keeps specifications at hand.
