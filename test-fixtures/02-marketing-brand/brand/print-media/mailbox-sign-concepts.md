# Mailbox Sign Concepts

## Design Option 1: Minimalist

```text
┌─────────────────────┐
│  SYMPHONY CORE      │
│  Suite 200          │
│  Mail Slot 12       │
└─────────────────────┘
```

## Design Option 2: Logo + Text

```text
┌─────────────────────┐
│  [LOGO]             │
│  Symphony Core      │
│  Suite 200          │
└─────────────────────┘
```

## Design Option 3: Vertical Layout

```text
┌───────────┐
│ SYMPHONY  │
│   CORE    │
│           │
│ Suite 200 │
│  Mail 12  │
└───────────┘
```

## Materials

- Acrylic (3mm thickness)
- Brushed aluminum
- Vinyl lettering

## Dimensions

- Width: 8 inches
- Height: 4 inches

## Color Scheme

Use brand colors:
- Background: Navy (#1E3A8A)
- Text: White (#FFFFFF)
- Logo: Full color
